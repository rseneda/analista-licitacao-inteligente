# Resumo Executivo

Este projeto apresenta a proposta de uma Plataforma de Licitação Inteligente, baseada em experiência prática na elaboração de ETP para contratação de software no setor público.

A solução visa reduzir ineficiências no processo de contratação, promovendo padronização, automação e melhoria na qualidade dos documentos.

Principais resultados esperados:

- Redução de tempo
- Melhoria da governança
- Apoio à tomada de decisão
- Mitigação de riscos

O projeto integra práticas de Product Owner, análise de requisitos, governança de TI e inteligência artificial.
