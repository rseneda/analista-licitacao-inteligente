# Papel do Product Owner

O Product Owner atua como responsável por:

- Definir backlog do produto
- Priorizar funcionalidades com base em valor
- Alinhar stakeholders
- Garantir aderência à legislação
- Validar entregas

## Atuação no projeto

- Priorização baseada em impacto
- Definição de requisitos do TR
- Integração com governança
