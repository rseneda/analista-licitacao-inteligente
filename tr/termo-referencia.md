# Termo de Referência

## 1. Objeto
Contratação de solução para gestão de contratações públicas de TI.

## 2. Justificativa
- Redução de tempo
- Padronização
- Transparência

## 3. Requisitos Funcionais
- Cadastro de demandas
- Geração de ETP
- Criação de TR
- Dashboard

## 4. Requisitos Não Funcionais
- Segurança (LGPD)
- Disponibilidade 99%
- Interface amigável

## 5. Critérios de Aceite
- Sistema deve gerar ETP completo
- Deve permitir edição
- Deve registrar histórico

## 6. SLA
- Suporte: 8h úteis
- Correção crítica: 24h


## 7. Modelo de Execução

A solução deverá ser implementada utilizando metodologia ágil, com backlog priorizado e entregas incrementais.

## 8. Critérios de Medição

- Redução de 30% no tempo de elaboração de documentos
- Redução de retrabalho
- Aumento da padronização

## 9. Governança

- Gestão baseada em backlog
- Reuniões de acompanhamento
- Indicadores de desempenho
